[ por que existe ]

Automação no Revit sem pedir para o usuário abrir terminal.

01

Add-in instalado

O instalador cria o arquivo `.addin` certo para a versão do Revit detectada.

02

MCP escondido

O launcher sobe o servidor local e mostra apenas a URL que o usuário precisa copiar.

03

Execução segura no contexto Revit

O código entra por fila, passa por `ExternalEvent` e roda no processo correto do Revit.

[ fluxo ]

O usuário vê uma URL. O sistema cuida do resto.

  1. 01

    ChatGPT chama a tool MCP.

  2. 02

    Servidor MCP local encaminha o comando para o add-in.

  3. 03

    Revit executa o C# via `ExternalEvent` e Roslyn.

  4. 04

    A resposta volta para o ChatGPT.

[ tutorial leigo ]

Instalação pensada para quem não programa.

1

Baixe o instalador

Use o `RevitMcpSetup.exe` publicado no GitHub Releases.

2

Abra o Revit

Reinicie o Revit depois de instalar para carregar o add-in.

3

Inicie o Revit MCP

Clique em `Start MCP`, depois em `Start Tunnel`.

4

Copie a URL

Cole a URL terminada em `/mcp` no ChatGPT.
