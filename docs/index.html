[ por que existe ]

Automação no Revit sem pedir para o usuário abrir terminal.

01

Add-in instalado

O instalador cria o arquivo `.addin` certo para a versão do Revit detectada.

02

MCP escondido

O launcher sobe o servidor local e mostra apenas a URL que o usuário precisa copiar.

03

Execução segura no contexto Revit

O código entra por fila, passa por `ExternalEvent` e roda no processo correto do Revit.

[ o que ele consegue fazer ]

O ChatGPT passa instruções para o Revit. O Revit executa de verdade.

Depois de conectado, você pode pedir para listar informações, criar elementos, editar parâmetros, contar objetos, gerar paredes, portas, janelas e outras automações usando a API do Revit.

listar

Níveis, paredes, vistas, famílias, tipos, parâmetros e quantidades.

criar

Paredes, linhas, pisos, instâncias de famílias e outros elementos via transação.

editar

Alterar parâmetros, nomes, tipos, posições e propriedades do modelo.

[ fluxo ]

O usuário vê uma URL. O sistema cuida do resto.

  1. 01

    ChatGPT chama a tool MCP.

  2. 02

    Servidor MCP local encaminha o comando para o add-in.

  3. 03

    Revit executa o C# via `ExternalEvent` e Roslyn.

  4. 04

    A resposta volta para o ChatGPT.

[ tutorial leigo ]

Instalação pensada para quem não programa.

1

Baixe o instalador

Use o `RevitMcpSetup.exe` publicado no GitHub Releases.

2

Abra o Revit

Reinicie o Revit depois de instalar para carregar o add-in.

3

Inicie o Revit MCP

Clique em `Start MCP`, depois em `Start Tunnel`.

4

Copie a URL

Cole a URL terminada em `/mcp` no ChatGPT.

[ teste de conexão ]

O primeiro teste só pergunta ao Revit qual projeto está aberto.

Depois de conectar o ChatGPT, peça para ele fazer um teste simples. Ele vai enviar o comando abaixo para o Revit. Se a resposta for o nome do seu arquivo, a conexão está funcionando.

mensagem para o ChatGPT teste seguro 
Use o Revit MCP para executar este teste:

return doc.Title;
Resultado esperado: o ChatGPT responde com o nome do projeto aberto no Revit.

[ exemplo avançado ]

Exemplo: pedir para o ChatGPT criar uma parede.

O usuário escreve um pedido simples. O ChatGPT transforma isso em C# compatível com o executor. O add-in abre uma transação no Revit, cria a parede e devolve uma confirmação.

pedido do usuário linguagem natural 
Crie uma parede reta de 10 metros no primeiro nível do projeto.
código enviado para run_revit_code C# Revit API 
using (var tx = new Transaction(doc, "Criar parede via Revit MCP"))
{
    tx.Start();

    var level = new FilteredElementCollector(doc)
        .OfClass(typeof(Level))
        .Cast<Level>()
        .First();

    var wallType = new FilteredElementCollector(doc)
        .OfClass(typeof(WallType))
        .Cast<WallType>()
        .First();

    var line = Line.CreateBound(new XYZ(0, 0, 0), new XYZ(10, 0, 0));
    Wall.Create(doc, line, wallType.Id, level.Id, 3, 0, false, false);

    tx.Commit();
}

return "Parede criada.";
